# SubGeo-Adam 技术报告（v1.1）

**全称**：Subspace-Geometric Anisotropic Damping for Continual LoRA Learning（SubGeo-Adam）

**定位**：冻结大模型主干，仅在 **LoRA 参数子空间** 上做持续学习；通过 **锚点处联合几何（梯度子空间 × 投影 Hessian）** 构造阻尼方向，并以 **非对称 Adam（动量通道与方差通道使用不同梯度）** 注入各向异性阻尼；多任务场景下配合 **预算约束与在线效用淘汰**。

**命名边界**：若与 Subspace Geometry 等“梯度子空间与遗忘”文献并列讨论，须在引言写明 **假设域（LoRA、任务序列、锚点协议）** 与 **贡献边界**，避免标题式挂靠。

**仓库入口**：实验排期与基线矩阵见 `docs/GeoAdam_Experiment_Plan.md`；**Phase0 烟雾测试**（可运行）见 `smoke/README.md`。

**核心叙事（问题—方法—判据，定稿）**：见 **`docs/CORE_SubGeo_Research_Narrative.md`**。

## 1. 问题设定与符号

### 1.1 持续学习协议

- 任务序列：T1 → T2 → … → TN。每阶段只关心 **当前任务损失** 最小化，同时希望 **旧任务性能** 少降。
- 参数：全模型权重冻结；可训练量为 **LoRA 增量** θ。将 θ 按模块展平为向量 **g ∈ R^D**（D 为 LoRA 总自由度，与 rank、层数有关，常为 10^4～10^5 量级）。

### 1.2 锚点

- 学完任务 A 后，在 LoRA 参数达到 **θ_A**（由早停或充分收敛判据固定；主实验只选一种协议，另一种放附录）。
- **几何信息在 θ_A** 处提取；新任务 B 训练时 θ 会离开 θ_A，这是所有“截面 Hessian / 旧任务梯度统计”方法的 **共同局限**，需用 **时效性曲线**（参数距锚点范数 vs 遗忘）回应审稿。

### 1.3 与 WeLore 等「梯度–Hessian 对齐」文献的并列方式

- **论文中须写清**：单任务近收敛动态下，梯度相关子空间与 Hessian 主导子空间 **可能高度对齐**，从而在锚点邻域使「联合」在数值上 **接近** Hessian-only。本文假设域为 **LoRA 持续学习与跨任务切换**：Task B 开始后一阶冲突方向随数据与当前 \(\theta\) 变化，锚点处缓存的截面 Hessian 在 Phase 内视为 **相对缓变**。
- **必须执行的诊断**：在 \(\theta_A\) 处报告 **\(V_{\mathrm{grad}}\)** 与 **仅 Hessian Top‑\(k\) 子空间** 的主夹角 / 重叠；在 Task B 训练 **动态段** 再测，正文或附录出图。

## 2. 阶段一：锚点校准与联合几何提取

### 2.1 目标

在 θ_A 得到：

1. **粗筛子空间** V_grad ∈ R^{D×r}（列正交）：描述与旧任务 / 冲突相关的 **梯度主导方向**。
2. **精修谱信息**：在 span(V_grad) 内得到 **Hessian 投影** 的特征分解，得到最终阻尼方向 V_k 与各向异性强度 λ_i、阻尼系数 γ_i。

### 2.2 Step 1a：梯度子空间（粗筛）

1. 在任务 A 的数据上，锚点 θ_A 附近（或就在 θ_A）采样 **B 个 mini-batch**，各算 **LoRA 梯度** g_b，展平为列向量，拼成矩阵 **G ∈ R^{D×B}**（或先对每列中心化，按实现固定一种）。
2. 对 G 做 **截断 SVD** 或 **随机 SVD**：取前 r 个左奇异向量，经 **QR** 得到 **V_grad**（D×r，列正交）。
3. **统计报告**（论文必须）：奇异值谱、能量累积曲线；说明 **B ≥ max(2r, 64)**（与实验计划一致，可调大做敏感性）。

### 2.3 Step 1b：投影 Hessian（精修）

1. 记对称 Hessian 近似为 **H**（对带 BN 等情形，实现上可用标准 Hvp 对称化约定；论文写明）。
2. 构造 **H_proj = V_grad^T H V_grad ∈ R^{r×r}**。
3. **实现**：对 j = 1…r，计算 **Hvp(V_grad 的第 j 列)**，再左乘 V_grad^T，得到 H_proj 的第 j 列。共 **r 次 Hvp**。
4. 对 H_proj 做 **特征分解**（r 很小，精确即可）：H_proj = Q diag(λ_1,…,λ_r) Q^T，Q 正交。
5. **回到全空间方向**：**V_k = V_grad Q**（D×r，列正交）。λ_i 即 **Ritz 值**（子空间内曲率指示）。

**负特征值与近零特征值**

- λ_i ≤ 0：默认 **γ_i = 0**（该 Ritz 方向不施加“凹方向阻尼”），论文解释与消融固定此规则。
- |λ_i| < λ_eps：视为数值近零，**γ_i = 0**。

### 2.4 Step 1c：阻尼系数与算子 P_d

对每个方向 i（对应 V_k 的第 i 列 v_i）：

- **γ_i = min(1, λ_i / τ)**，τ > 0 为超参（中位数 / 分位数 / 网格搜索，主文给敏感性曲线）。
- **软阻尼算子**（仅通过矩阵–向量积实现，**禁止**显式构造 D×D 矩阵）：

**P_d g = g − V_k ( γ ⊙ (V_k^T g) )**

其中 ⊙ 为逐元素乘，γ = (γ_1,…,γ_r)^T。

### 2.5 成本

- **单次 Hvp** 与 **单次对 LoRA 的反传** 同阶，且必须经过 **完整前向–反向图**，记 **T_full**。
- Step 1a：**约 B 次 T_full**（收集梯度）。
- Step 1b：**约 r 次 T_full**（r 次 Hvp）。
- **合计约 (B + r) 次 T_full**；与“全空间 Lanczos 取 Top-k”常用 **m 次 Hvp** 往往 **同量级**，须在论文中用 wall-clock 与表格并列 **B、r、m**。

## 3. 阶段二：非对称 Adam（仅 LoRA 参数）

### 3.1 递推（与标准 Adam 对齐的记号）

对 LoRA 的每个标量参数分量下标用 j；实现上可对展平向量整体写，此处给出 **与 AdamW 兼容** 的写法要点。

- **两路梯度**：
  - **动量路**：g_m = P_d g（对当前反传得到的 g 先做 P_d）。
  - **方差路**：g_v = g（原始梯度）。
- **一阶矩、二阶矩**（与 Adam 一致，β1、β2、ε 全实验固定）：
  - m ← β1 m + (1−β1) g_m
  - v ← β2 v + (1−β2) (g_v ⊙ g_v)   （逐元素平方）
- **权重衰减（AdamW）**：对 LoRA 权重做 **解耦衰减** 时，与基线完全一致；**不得**在动量路偷偷混入未声明的 L2。
- **参数更新**：θ ← θ − η · m / (√v + ε)（逐元素；或采用与 PyTorch AdamW 相同的 bias correction 规则，**全方法统一**）。

### 3.2 与“尺度不变性”叙事的关系（表述约束）

- **全局常数缩放** g → c g 时，对称使用同一 g 的 m、v 会呈现经典尺度行为；**非对称** g_m ≠ g_v 时，**不能**用一句“完全破解”带过；正文应写为：**在固定坐标系下，危险方向上的等效步长受 γ 与 v 的共同调制，且与仅缩放 g 再喂 Adam 不同**。
- **Sym-Geo（消融）**：g_m = g_v = P_d g，用于对照“仅阻尼无分路”。

### 3.3 工程路径

- **主路径**：对 LoRA 参数组使用 **自定义优化器步**（或等价：手写 step 仅遍历 LoRA）；其余参数（若有）用 **标准 AdamW**。
- **理由**：标准 AdamW **单次 step 只读一份 grad**，无法在不做内核改动的情况下实现真正的 g_m ≠ g_v；这与先前“纯 hook 等价实现非对称”的结论一致。

## 4. 阶段三：多任务、预算与在线效用淘汰

### 4.1 全局保护基

- 每学完一个任务，在对应锚点提取一组 (V_k, λ, γ) 或等价信息；将方向并入 **V_global**（列正交化用 **修正 Gram–Schmidt 或 QR**，禁止朴素 GS 无重正交）。
- **硬预算**：全局最多保留 **K_max = floor(ρ · r_lo)** 个保护方向（ρ 默认 0.8；与 rank 关系写死并做敏感性）。

### 4.2 效用（定稿形式 + 在线更新）

**设计理由**：单一乘积 \(U_i=\lambda_i(1+\alpha|g|)\) 在 \(|g|\to 0\) 时仍可能因大 \(\lambda_i\) 占用预算，与「优先保留 **当前有冲突且陡峭** 的方向」不一致；故将 **曲率** 与 **当前冲突强度** 分项组合。

**默认一版（可消融）**

- 在 Task B 训练第 t 步（或每 T 步），用 **当前参数 θ(t)** 与 Task B 的 batch 计算 LoRA 梯度 g_B(t)（展平）。
- 对每个保护方向 i（对应 V_global 的第 i 列 u_i）：
  - **冲突强度**：c_i(t) = | u_i^T g_B(t) |。
  - **效用**（示例）：**U_i(t) = λ_i · ReLU( c_i(t) − ε )**，ε ≥ 0 为小常数或按 |c| 的分位数自适应（固定一种写进附录）。
  - 含义：**当前几乎不走的方向**（c_i ≤ ε）效用为 0，在“淘汰最小 U”规则下会优先释放预算；**高曲率且当前强冲突** 的方向 U 大，优先保留。

**你们提出的在线更新**（推荐写进主方法）：

- 每 T 步（或每 epoch）用当前 g_B(t) 重算 {U_i(t)}；若 |V_global| > K_max，则 **删除 argmin U_i**（或一次删多个至满足预算）。
- **λ_i 的来源**：默认 **锚点提取后固定**（低成本、叙事清晰）；若要做“二阶也慢更新”，必须另开消融并报告开销。

**与“预探测悖论”的关系**

- 一次性在 θ_A 用 Task B 数据算梯度，确实与训练中 θ 偏移不一致；**在线 U_i(t)** 缓解该问题，代价为每 T 步额外统计（通常远小于训练主体开销，需写入开销表）。

**仍未在数学上“证最优”**

- “淘汰 min U”是 **预算约束下的启发式**；正文写 **设计目标**（与 `GeoAdam_Experiment_Plan.md` 中效用与淘汰条款对齐），避免声称全局最优淘汰定理。

## 5. 消融链（修订后假设，用实验钉死，不预写死强弱）

以下编号与实验脚本配置 ID 对齐。


| ID  | 配置                                                                    | 目的                                  |
| --- | --------------------------------------------------------------------- | ----------------------------------- |
| (1) | Vanilla LoRA + AdamW                                                  | 基线                                  |
| (2) | **梯度子空间投影 + AdamW**（仅改方向：类似 SGP / O-LoRA 思想，实现需与 CODE-CL 区分：后者多在激活空间） | 验证“**改方向**”本身是否有帮助                  |
| (3) | **同向标量缩放/阻尼梯度 + AdamW**（m、v 同用缩放后的 g；即路径 B / Grad-Damp）               | 验证“**主要改幅度、且 m/v 对称**”的极限           |
| (4) | Symmetric Geo-Adam（g_m = g_v = P_d g）                                 | 与 (3) 对照，检验 **P_d 与 Adam 耦合** 的非平凡性 |
| (5) | **Asymmetric SubGeo-Adam（Ours）**                                      | 主方法                                 |


**假设性结论链（待数据验证，论文中勿写死为定理）**

- 期望 **(5) 优于 (4)**：支持 **非对称分路** 的必要性。
- **(2) 可能优于 (1)**：方向干预在 CL 中常见有效。
- **(3) 相对 (1)**：未必无效；若 (3)≈(1) 而 (5) 仍强，叙事侧重 **非对称 + 曲率**；若 (3) 已明显优于 (1)，则叙事侧重 **在 (3) 之上 (5) 仍有增益**。

**关于 SGP 与 Adam 的澄清（写入 Related Work 一段）**

- 若仅做 **全局标量** g → α g，Adam 的稳态更新尺度行为与“仅改学习率”有相似之处；但 **正交投影 / 子空间投影** 改变的是 **各坐标分量相对比例**，一般 **不能** 被“等价于乘常数”吸收。
- 具体结论以 **你们实现的 (2)(3)** 在相同 P_d 或相同子空间设置下的曲线为准。

## 7. 实现清单（PyTorch 侧）

1. **LoRA 展平 / 还原**：`named_parameters` 过滤 LoRA；`flatten` 与 `view_as` 成对单元测试。
2. **Hvp**：`torch.autograd.functional.hvp` 或双反传；在 **混合精度** 下固定 autocast 策略并写入复现包。
3. **SVD / QR**：`torch.linalg.svd` + `torch.linalg.qr`；大 B 用随机 SVD（固定种子）。
4. **P_d @ g**：`g - V @ (gamma * (V.T @ g))`，**O(D·k)**，禁止构造 P_d 矩阵。
5. **Geo 优化器**：仅遍历 LoRA 参数；`state` 存 m、v；与 **AdamW decoupled weight decay** 对齐。
6. **在线淘汰**：每 T 步钩子；与 DDP 同步时 **V_global、γ、λ 各 rank 一致**（广播或 rank0 计算）。

## 9. 设计约束与实现注意事项

1. **U_i = λ_i · ReLU(c_i − ε)**：当 c_i ≤ ε 时 U_i=0，这些方向会最先被淘汰；若后续优化 **突然激活** 该方向且 λ_i 曾很大，存在 **旧任务风险**。**缓解**：ε 不要过大；或引入极小 **λ 正则项** U_i = λ_i·ReLU(c_i−ε) + η·1_{λ_i large} 等（仅作附录消融），不可 silent 改主方法。
2. **消融链 (4) ≈ (2)**：不一定成立；(4) 是 **P_d 后对称 Adam**，(2) 是 **投影器未必等于 P_d**。论文里用 **并列对照** 而非链条等式。
3. **层级别启发式**：若正文采用“底层 ρ 大、顶层 ρ 小”，须 **固定规则 + 敏感性**，否则审稿归为 ad-hoc；复杂 per-layer 仍建议附录。

## 10. Phase0 烟雾测试（验证实现，而非 CL 优劣）

**目的**（先于 LLM）：在可控 **二次型** 上做 **数值与分路自检**——确认 `P_d` 实现正确、`sym` 与 `asym` 在 **γ 非零** 时第一步可区分、**弱阻尼（γ→0）** 时各 mode 与 `vanilla` 一致、`reverse` 与 `asym` 的 **‖V^T m‖** 可区分。

**刻意不测**：不在此阶段要求 **asym 终 loss 优于 sym**。在纯凸二次型上 **强阻尼** 会压制沿主曲率方向的必要更新，劣于 vanilla 属正常；强弱 Pareto 留给真实持续学习实验。

**过线（脚本断言）**：见 `smoke/run_smoke.py` 内四个 `test_`*；退出码 0 即通过。

**运行**：仓库根目录执行 `py -3 smoke/run_smoke.py`（需已安装 `torch`）；说明见 `smoke/README.md`。

**Phase1 玩具 CL**（两任务 + 锚点 + vanilla vs asym）：`experiments/phase1/run_toy_cl.py`。

**几何提取实现**：`src/subgeo/geometry.py`（`collect_grad_flat_matrix`、`subspace_from_G`、`projected_hessian`、`build_gamma`）；**锚点联合一步封装** `src/subgeo/joint_geometry.py`（`extract_joint_vk_gamma`）；HF 判别脚本 `experiments/phase1/run_joint_geometry_cl.py`。自检脚本 `experiments/phase1/verify_geometry.py`。
