## 1. 问题与目标

**场景**：大模型主干冻结，用 **LoRA** 做多阶段持续学习；核心困难是 **灾难性遗忘** 与 **新任务可塑性** 的折中。

**目标**：在 **不显式改写总损失** 的前提下，把「旧任务在参数空间中的危险几何」编码进 **优化器内部递推**：对动量与二阶矩使用 **不同** 的梯度映射，使 **陡峭 Hessian 方向** 上的步长被 **各向异性** 地抑制，而不过度损伤平坦方向与可学方向。

**命名**：对外主名 **SubGeo-Adam**（Subspace-Geometric Anisotropic Damping for Continual LoRA Learning）。与 Subspace Geometry、WeLore 等文献并列时，须在论文中写明 **假设域（LoRA、任务序列、锚点协议）** 与 **本文贡献边界**。

## 2. 方法总览（三阶段）

### 2.1 阶段一：锚点与联合几何

**锚点 \(\theta_A^\*\)**：在旧任务数据上训练至协议收敛点；**所有截面几何在该点提取**。新任务训练时 \(\theta\) 会漂移，这是截面方法的 **共同局限**，用 **时效性曲线**（距锚点范数 vs 遗忘）与 **动态段诊断** 在实验中报告。

**联合几何（默认协议）**：

1. **粗筛（一阶）**：在 \(\theta_A^\*\) 上采集 **B** 个 mini-batch 的 LoRA 梯度，拼成 \(G\in\mathbb{R}^{D\times B}\)，截断 SVD + QR 得 **\(V_{\mathrm{grad}}\in\mathbb{R}^{D\times r}\)**。要求 **\(B\ge \max(2r,64)\)**（主实验写死一种，敏感性放大 B）。
2. **精修（二阶）**：在 \(\mathrm{span}(V_{\mathrm{grad}})\) 上构造 **\(H_{\mathrm{proj}}=V_{\mathrm{grad}}^\top H V_{\mathrm{grad}}\)**；对 \(V_{\mathrm{grad}}\) 的每一列做一次 **Hvp**，共 **r** 次；对 \(r\times r\) 对称阵特征分解得 Ritz 值 \(\lambda_i\) 与特征向量矩阵 \(Q\)；**\(V_k = V_{\mathrm{grad}} Q\)**（列仍正交）。
3. **阻尼**：\(\gamma_i=\min(1,\lambda_i/\tau)\)（\(\lambda_i\le 0\) 或数值近零则 \(\gamma_i=0\)）；**\(P_d g = g - V_k(\gamma\odot(V_k^\top g))\)**；禁止显式构造 \(D\times D\) 矩阵。

**成本口径（论文与实验统一）**：单次 Hvp 与 **一次完整前向+反向** 同阶，记 **\(T_{\mathrm{full}}\)**。阶段一合计约 **\(B+r\) 次 \(T_{\mathrm{full}}\)****，与全空间 Lanczos 取 Top‑\(m\) 的 **\(m\) 次 Hvp** 同量级时，用 wall-clock 表并列报告 **B、r、m**。

**诊断（正文或附录必含）**：锚点处 **\(V_{\mathrm{grad}}\)** 与 **仅 Hessian Top‑\(k\) 子空间** 的主夹角 / 重叠；Task B 训练 **动态段** 再测，用于说明联合信号的价值落在 **跨任务阶段** 而非单点口头论证。

### 2.2 阶段二：非对称 Adam（仅 LoRA）

- **\(g_m = P_d g\)**（入动量），**\(g_v = g\)**（入方差）；\(\beta_1,\beta_2,\epsilon\) 与 **AdamW 解耦权重衰减** 与基线严格一致。
- **对称对照（消融）**：\(g_m=g_v=P_d g\)，用于隔离「仅 \(P_d\) 对称进入 m、v」与「非对称分路」的差异。

**与尺度行为的表述约束**：不宣称「全局破解尺度不变性」；在固定坐标系下写明 **\(g_m\neq g_v\)** 时分子分母受 \(\gamma\) 与 \(v\) 的 **联合** 调制，与「仅对称缩放梯度再喂 Adam」可区分，由消融数据支撑。

### 2.3 阶段三：预算与在线效用

**全局保护基**：每任务结束时将新提取方向经 **修正 Gram–Schmidt 或 QR** 并入 \(V_{\mathrm{global}}\)；硬预算 **\(K_{\max}=\lfloor \rho\cdot r_{\mathrm{lo}}\rfloor\)**（\(\rho\) 默认约 0.8，做敏感性）。

**效用（定稿默认）**：将 **曲率项** \(R_i=\phi(\lambda_i)\) 与 **冲突项** \(C_i=\psi(\hat c_i)\)（\(\hat c_i\) 为当前任务梯度在方向 \(i\) 上的投影强度）组合为 **\(U_i=R_i+wC_i\)**。其中 \(\psi\) 采用 **ReLU(\(\hat c_i-\varepsilon\))** 等形式，使 **当前几乎不激活的方向** 在「按 \(U\) 最小淘汰」规则下 **优先释放预算**；避免单一乘积 \(\lambda_i(1+\alpha|g|)\) 在 \(|g|\to 0\) 时仍被大 \(\lambda_i\) 占满预算的语义问题。

**在线更新**：每 **\(T\)** 步用当前 \(\theta(t)\) 与当前任务 batch 重算 \(\hat c_i\)，更新 \(U_i\)；超预算则删除 \(\arg\min U_i\)（或批量删至满足预算，协议写死一种）。\(\lambda_i\) **默认锚点固定**；若对 \(\lambda\) 做慢更新，单开消融并报开销。

## 3. 消融与基线（证明链）

**机制消融（与 RQ1 对齐）**：

| ID | 配置 | 在证明链中的角色 |
|----|------|------------------|
| (1) | Vanilla LoRA + AdamW | 下界 |
| (2) | 梯度子空间投影 + AdamW | 「改方向」类（与 SGP/O-LoRA 思想可比） |
| (3) | 同向标量缩放 / Grad-Damp + AdamW | 「主要改幅度且 m、v 对称」 |
| (4) | Symmetric：\(g_m=g_v=P_d g\) | 与 (3) 对照，检验 **\(P_d\) 与 Adam 耦合** |
| (5) | Asymmetric SubGeo-Adam | 主方法 |

**读链（假设性，由数据验证）**：期望 **(5) 优于 (4)**；**(2)** 相对 **(1)** 可提升；**(3)** 相对 **(1)** **不预设强弱**——若 **(3)** 已强而 **(5)** 仍增益，叙事强调 **非对称通道**；若 **(3)** 弱而 **(5)** 强，叙事强调 **曲率 + 非对称**。

**信号源消融（与 RQ2 对齐）**：Hessian-only、Grad-only、Joint（主）；并报告 §2.1 所述夹角诊断。

**预算消融（与 RQ3 对齐）**：无预算堆叠（展示维度坍塌）、\(K_{\max}\)+曲率淘汰、\(K_{\max}\)+效用 \(U_i\)；以及 **\(w=0\)**（纯曲率排序）等对照。

**P0 基线矩阵**：Vanilla、强 L2/WD、EWC（或 EWC-LoRA）、O-LoRA/OPLoRA、SOAP+LoRA、**CODE-CL**、**SGP**、Ours（Joint+Asym）。相关工作表须写清 **操作空间（激活 vs 参数）**、**统计阶（一阶 vs 二阶）**、**是否改写优化器 m/v**。

## 4. 评价指标与「是否值得推进」的判据

- **帕累托**：旧任务遗忘 proxy vs 新任务性能。
- **机制**：\(\|V_k^\top m\|_2\)、\(\|P_d g\|/\|g\|\)、Ritz 值随训练漂移（可选）。
- **成本**：每阶段 **\((B+r)\,T_{\mathrm{full}}\)**、在线淘汰每 \(T\) 步开销、显存峰值。

**初步判断「值得继续投入 Phase2/论文」的最低门闸（工程可执行）**：

1. 在 **Joint + Asym（5）** 与 **对称/Grad-only** 对照上，**至少一种** 主设定下 **帕累托或遗忘 proxy** 相对 **(1)(4)** 有 **稳定**（多种子）优势；且  
2. **Joint** 相对 **单信号源** 不系统性更差，并有 §2.1 诊断支撑叙事；且  
3. 成本曲线落在预算内（单卡 48GB 路线可接 QLoRA）。

未过门闸则 **收窄 claim 或调整 \(\tau,\rho,B,r,\phi,\psi,w\)**，而非继续堆工程烟雾。
